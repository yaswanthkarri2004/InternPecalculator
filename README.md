## Run Locally

Clone the project

```bash
  git clone https://github.com/yaswanthkarri2004/InternPecalculator.git
```

Go to the project directory

```bash
  cd InternPecalculator
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
